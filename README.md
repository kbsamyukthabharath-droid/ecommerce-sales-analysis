# E-Commerce Sales Analysis

## Overview
This project focuses on cleaning, analyzing, and visualizing e-commerce sales data using Python. The objective is to transform raw sales data into meaningful business insights through data preprocessing, exploratory data analysis (EDA), and visualization techniques.
The dataset contains information related to customer orders, products, sales, regions, payment methods, and customer ratings. Various data quality issues such as missing values, duplicate records, inconsistent data formats, and outliers were identified and addressed during the analysis process.
